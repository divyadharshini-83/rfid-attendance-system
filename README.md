# RFID Attendance System

This project is an RFID based smart attendance system using Arduino and MFRC522 RFID reader.

## Hardware Used
- Arduino Uno
- MFRC522 RFID Reader
- RFID Cards
- LEDs
- Buzzer

## Features
- Automatic attendance marking
- Prevents duplicate attendance
- LED and buzzer indication
- Serial monitoring

## Technologies
- Arduino C++
- SPI Communication
- RFID

## How it Works
1. RFID card is scanned
2. UID is read by MFRC522
3. UID is compared with stored database
4. Attendance is marked
5. LED and buzzer provide feedback
